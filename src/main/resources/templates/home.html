LTI 1.0 provider endpoint

The LTI 1.0 provider is at 

http://localhost:8080/tsugi
If you navigate to this directly, you will get an error that says, "Request is not an LTI request". You must launch this using LTI - perhaps using 
http://localhost:8888/tsugi/dev.php
and put in something like "http://localhost:8080/tsugi" as the launch url and press "Launch".
